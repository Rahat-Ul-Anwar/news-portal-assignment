Some Important Question on JS

1. Difference between var, let and const keywords in JavaScript?

VAR : The var is the oldest keyword to declare a variable in JavaScript. The scope of the var keyword is the global or function scope.

LET : The let keyword is an improved version of the var keyword. The scope of a let variable is only block scoped.

CONST : The const keyword has all the properties that are the same as the let keyword, except the user cannot update it.When users declare a const variable, they need to initialize it, otherwise, it returns an error.

2. Difference between Regular Function and Arrow Function in JavaScript?

Regular Function : You can call both function declaration and expression as Normal/Regular Function

Arrow Function : Arrow function is introduced in ES6 and also known as fat arrow function.

2. Difference between Map, Filter, ForEach and Find in JavaScript?

MAP() : Map function is itterable. when it itterates the array it takes the each element and process something and return the array

FILTER() : Filter function is also itterable and when it itterates the array it takes the each element and mathches some specific condition then return the array.

FOR EACH () : Fore each function is also itterable and when it itterates the array it takes the each element and fulfilled specific condition but didnt return any array.

FIND() : Find function is also itterable and when it itterates the array it takes the each element and fulfilled specific condition then return the first element of the array.
